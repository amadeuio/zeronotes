# Zeronotes

A fullstack notes app with secure backend, authentication, and data persistence.

> **Work in progress.** Backend integration underway.

## What's Working

- Custom drag-and-drop masonry grid
- Real-time filtering and search
- Express API with PostgreSQL
- CRUD operations for notes and labels
- Many-to-many relationship between notes and labels
- Runtime type validation with Zod
- JWT-based authentication

## What's Next

- User authentication
- End-to-end encryption
- Deploy backend + frontend

## Tech Stack

**Frontend**

- React 19
- TypeScript
- Zustand + Reselect
- Tailwind CSS

**Backend**

- Node.js + Express
- TypeScript
- PostgreSQL
- pg (PostgreSQL client)
- Zod (Runtime validation)
- Argon2 (Password hashing)
- Jose (JWT tokens)
